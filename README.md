Final Project to create a high fidelity prototype web design for university lab: Human and Computer Interaction 
cryPIto is a instructor assigned website to monitor cryptocurrencies.
The design has 5 pages: Home, Cryptocoins, Learn More(FAQ), Benefits, and Register
The home page shows the sign up button, brief explanation of what cryptocurrency is, and what well known cryptocurrencies are monitored.
The cryptocoin page lists 17 cryptocurrencies with a view more hyperlink at the end
The FAQ page has a list of general FAQs questions and implements a scrolling hyperlink that automatically scrolls to the answer when the question on the left is clicked.
The benefit page simply lists the benefits one can get when they join "cryPIto"
lastly the register page is a sign up/in page that links to the cryptocoin page when all registration validations were satisfied
